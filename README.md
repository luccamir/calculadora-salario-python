# Calculadora de Salário Líquido 2026

Programa em Python que calcula os descontos de INSS e
Imposto de Renda 2026 a partir do salário bruto informado.

## O que o programa faz
- Recebe o salário bruto do usuário
- Calcula o desconto do INSS (tabela progressiva 2026)
- Calcula o desconto do IR (isenção até R$ 5.000 - nova regra 2026)
- Exibe o salário líquido final

## Tecnologias
- Python 3

## Como executar
python atividade_python_inss_ir.py
